# MovieWorld App

## Author

Jeff Mbithi

## Description

MovieWorld is a web application that allows users to browse for movies and series. Users can search for movies, filter by genre, and view detailed descriptions of the movies. The project is built using JavaScript, HTML, and CSS, with a JSON server providing backend functionality.

## Project Setup Instructions

### Installation Steps

1. Clone the repository:

   git clone https://github.com/Jeff-m9/MovieWorld-App-Project.git

2. Navigate to the project directory:

   cd MovieWorld

3. Install JSON Server (if not already installed):

   npm install -g json-server

4. Start the JSON Server:

   json-server --watch db.json --port 3000

5. Open `index.html` in your browser or use Live Server (if using VS Code).
